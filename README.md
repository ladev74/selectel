## Сборка плагина
```text
Makefile
В корне репозитория выполните команду:
```
```bash
make all
```

```text

Она выполнит три команды:
build - сбилдит бинарный файл командой golangci-lint custom -v
install - перенесет полученный файл в директорию GOBIN
copy_config - скопирует конфиг для линтера из корня репозитория в домашнюю директорию
```



Ручной способ:

В корне репозитория выполните команду:
```bash
golangci-lint custom -v
```
```text
после выполнения вы получите бинарный файл под названием custom-gci, по желанию можно переименовать,
это стандартный golanci-lint с подключенным плагином.
Затем переместите файл .golangci.yaml из корня репозитория или в свою домашнюю директорию (тогда конфиг станет глобальным)
или в любой другой проект где вы захотите пользоваться плагином. После этого для удобства скопируйте бинарный файл
в директорию go/bin чтобы линтер можно было вызывать из любого каталога
```

## Примечания
```text
В файле .golangci.yaml уже есть настройки для плагина, можно при желанию выключить дефолтные плагины,
для самого плагина выключать/включать правила и добавлять паттерны для проверки чувствительных данных
```
